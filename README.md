# Matter Capture Tool

Tool for Minetest that captures exact blocks (nodes, not their standard drops), including sources (water, lava), with survival durability (331 uses), creative mode safe and container safety.

## Crafting
default: "Gold Ingot", "Diamond", "Mese Crystal Fragment",
         "Locked Chest", "Mese Crystal", "Obsidian Glass",
         "Steel Ingot", "Diamond", "Mese Crystal Fragment"
